## 项目经历

### Vue3 Admin System 后台管理系统

项目链接：https://vue3-admin-system.vercel.app/login  
技术栈：Vue3、Vite、Pinia、Vue Router、Element Plus、Axios、ECharts、Node.js、Express、JWT

项目描述：  
基于 Vue3 + Express 搭建的前后端分离后台管理系统，围绕登录鉴权、用户管理、数据看板和个人设置等典型后台业务场景，实现从前端页面、状态管理、接口封装到后端鉴权接口的完整闭环。

核心工作：
- 使用 Vue3 Composition API + Element Plus 搭建登录页、后台布局、用户管理、Dashboard、个人设置等页面，完成表单校验、弹窗交互、分页筛选和状态提示。
- 基于 Pinia 管理登录用户、Token 和用户列表状态，并结合 localStorage 做登录态持久化，刷新页面后可恢复基础登录信息。
- 封装 Axios 实例，在请求拦截器中统一注入 `Authorization: Bearer token`，在响应拦截器中处理 401 未登录状态并清空本地登录态。
- 使用 Vue Router `beforeEach` 实现路由守卫，未登录用户无法访问后台页面，已登录用户访问登录页时自动跳转到后台首页。
- 基于 Express + JWT 实现登录接口和鉴权中间件，对用户管理、数据看板、个人设置接口进行 Token 校验。
- 实现用户管理模块的新增、编辑、删除、模糊搜索、角色筛选和分页展示，并区分 Admin/User 的操作权限。
- 使用 ECharts 实现用户总数、活跃用户、非活跃用户、管理员数量的数据看板，并在用户数据变化后联动刷新图表。

项目收获：
- 熟悉了前端后台管理系统的基础业务结构，理解登录鉴权、路由拦截、接口封装、状态管理和权限控制之间的协作关系。
- 具备独立完成一个 Vue3 前后端分离项目从开发、构建到线上部署的实践经验。
